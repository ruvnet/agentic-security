# Aider VSCode Extension

AI pair programming with Aider in VSCode.

## Features
- Streamlit-based UI
- Git integration
- Model switching
- Temperature control
- File management

## Development
1. npm install
2. pip install -r src/python/requirements.txt
3. F5 to start debugging

## Usage
1. Command Palette -> "Start Aider Chat"
2. Use the chat interface in the panel
